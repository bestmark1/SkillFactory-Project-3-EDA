# Проект 3: Разведывательный анализ данных (EDA)

## Описание проекта
Данный проект выполнен в рамках обучения в SkillFactory. Главная цель проекта — проведение полноценного разведывательного анализа (EDA) датасета отзывов об отелях (Booking.com) и подготовка данных для построения модели машинного обучения `RandomForestRegressor`, которая будет предсказывать рейтинг отеля по отзыву клиента.

## Выполненные задачи:
1. **Очистка данных:**
   - Обработка пропущенных значений (географические координаты `lat`, `lng` заполнены медианными значениями для сохранения тестовой выборки).
   - Преобразование строковых признаков (например, `days_since_review`) в числовой формат типов integer.

2. **Генерация новых признаков (Feature Engineering):**
   - Выделена "Сезонность": извлечены месяц и год из даты отзыва.
   - Сгенерированы бинарные флаги-индикаторы тональности отзывов (`is_no_negative`, `is_no_positive`), так как наличие или отсутствие жалоб напрямую коррелирует с итоговой оценкой.
   - Рассчитана историческая "Истинная репутация отеля" (Target Encoding), дающая алгоритму понимание исторического качества обслуживания в каждом конкретном отеле.

3. **Отбор признаков (Feature Selection):**
   - Был проведен тест значимости признаков (`f_regression` / ANOVA) и анализ мультиколлинеарности. Вся текстовая информация была удалена (переведена в числа), чтобы алгоритм "Случайного леса" смог корректно на ней обучиться.

## Результаты моделирования
Был обучен базовый алгоритм `RandomForestRegressor` со 100 деревьями. 
В результате: средняя абсолютная процентная ошибка (**MAPE**) на валидационной выборке составила **~13.58%**. Этот результат означает, что модель ошибается всего на 1.3 балла из 10, что является отличным бейзлайн-результатом. Финальный прогноз на тестовых данных был зафиксирован в формате соревнования Kaggle в `submission.csv`.
